# About-Sass

Sass使用方法

1. sass基于Ruby语言开发而成，因此安装sass前需要安装Ruby。(注:mac下自带Ruby无需在安装Ruby!)

[Ruby下载](https://rubyinstaller.org/downloads/)

2.安装Ruby

勾选Add Ruby executables to your PATH添加到系统环境变量

3.打开控制台

 1.ruby -v 检验ruby是否安装成功
